

**所有联网程序都需要进行网络通信**
计算机只有一个物理网卡，而且同一个局域网中，网卡的地址是必须唯一的。
网卡是通过唯一的ip地址来进行定位的。

*服务器：24小时不关机的电脑*

 - **ip地址用来定位计算机**
 - **端口号用来定位具体的应用程序**
 - (所有需要通信联网的软件必须要有端口号)
 - 端口号的范围从0~65536之间
 - 在计算机中有一些默认端口号，最好不要使用，例如http服务的80
 - 我们在开发中使用一些好记的就行了
