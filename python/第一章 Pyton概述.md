# 1、初始Python
## 1.1、Python语言简介
<font color="red">解释型，面向对象</font>，被广泛应用于处理系统管理任务和科学计算
## 1.2、特点

 1. 简单
 2. 高级
 3. 面向对象
 4. 可扩展性
 5. 免费和开源
 6. 可移植性
 7. 丰富的库
 8. 可嵌入性

## 1.3、缺点

 1. 速度慢
 2. 代码不能加密
 3. 多线程支持不好

## 1.4、应用范围

 1. 操作系统管理
 2. 科学计算
 3. Wen应用
 4. 图形用户页面开发
 5. 其他：如游戏开发，编写游戏的逻辑
